Pouzite technologie

Pro projekt pouzite nasledujici technologie:

  • Spring MVC
  • Spring Security Module (Autorizace s ruznymi roly uzivatelu)
  • Hibernate ORM (JPA provider)
  • Maven
  • MySQL database
  • TomEE Plume
